Canopy - プライバシーポリシー

最終更新日: 2026年3月22日

データ収集について

Canopy はユーザーのデータを一切収集しません。

ネットワーク通信

Canopy はインターネットに接続しません。すべての処理はお使いの Mac 上でローカルに実行されます。

ファイルアクセス

Canopy はユーザーが選択した SwiftUI プロジェクトフォルダ内のファイルを読み取り専用でアクセスします。ファイルの内容を外部に送信したり、変更したりすることはありません。

第三者への提供

個人情報を含むいかなるデータも第三者に提供することはありません。

お問い合わせ

本ポリシーに関するお問い合わせは GitHub Issues までお願いします。
